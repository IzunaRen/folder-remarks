# 工作区备注（Folder Remarks）

为工作区内任意文件或文件夹设置“备注名”，并在侧边栏的备注树里直观展示，支持右键菜单与命令面板快速管理。

## 功能

- 设置/修改/清除备注：对任意文件或文件夹保存一个可搜索的备注名
- 备注树（侧边栏）：以工作区文件树形式展示资源，并在条目右侧展示备注（如 `【无】`）
- 资源管理器/编辑器右键：在 Explorer 或编辑器标题栏右键快速设置/删除备注
- 管理备注（命令面板）：QuickPick 列表检索并支持一键编辑/删除
- 追踪当前文件：可选自动在备注树里定位当前编辑器打开的文件
- 文件操作：在备注树内支持打开/侧边打开/在资源管理器定位/复制路径/重命名/删除/新建文件(夹)

## 使用方式

### 方式一：资源管理器/编辑器右键（推荐）

- 右键任意文件或文件夹 → 设置备注
- 右键任意文件或文件夹 → 删除备注

### 方式二：侧边栏「备注」视图（备注树）

- 在活动栏打开「备注」视图
- 点击条目可打开文件
- 在条目右键菜单中使用：打开/侧边打开/定位/复制路径/重命名/删除/新建等操作

### 方式三：命令面板

在命令面板输入关键词 “Remark” 或 “备注”，常用命令包括：

- 设置备注
- 管理备注
- 追踪当前文件
- 刷新

## 快捷键

- 设置备注：`Alt+E`（macOS 通常为 `⌥E`）。在资源管理器或编辑器中使用，可快速对当前选中资源/当前文件执行“设置备注”（命令：`folderRemarks.setRemark`）
- 自定义：打开 VS Code 的 Keyboard Shortcuts（键盘快捷方式）中搜索 `folderRemarks.setRemark`，按需修改为你习惯的组合键

## 配置

- `folderRemarks.language`
  - `auto`：跟随 VS Code 显示语言
  - `en`：英文
  - `zh-cn`：简体中文
- `folderRemarks.autoRevealActiveFile`
  - `true`：自动在备注树中定位当前编辑器打开的文件
  - `false`：关闭自动追踪

示例配置见 [examples/settings.json](./examples/settings.json)

## 数据存储

备注数据存储在工作区根目录下的文件中：

- `.FolderRemarks/folder-remarks.json`

## 兼容性说明

- 建议在“单文件夹工作区”使用（当前存储与部分路径解析以第一个工作区文件夹为准）
