# 工作区文件夹备注（Trae 插件）

为当前工作区内的任意文件或文件夹添加可持久化的自定义备注名称，并提供侧边栏与命令面板的快速管理入口。

## 功能

- 为工作区内任意文件或文件夹添加/编辑/删除备注名称（持久化存储）
- 侧边栏视图：可视化列表、实时搜索、快捷操作（打开/编辑/删除）
- 命令面板：管理备注（QuickPick 实时搜索/筛选）
- 资源管理器右键菜单：对文件/文件夹直接增删改备注
- 工作区切换、窗口重载后自动还原

## 使用方式

### 方式一：资源管理器右键（推荐）

在资源管理器中对任意文件或文件夹右键：

- 添加备注（文件/文件夹）
- 编辑备注（文件/文件夹）
- 删除备注（文件/文件夹）

### 方式二：侧边栏视图

在活动栏中打开「备注」视图，支持：

- 搜索框实时过滤
- Add 按钮新增备注
- 每条备注支持 Open / Edit / Delete

### 方式三：命令面板

打开命令面板，执行：

- 文件夹备注：添加文件夹备注
- 文件夹备注：管理文件夹备注

在「管理」列表中：

- 直接输入关键词实时搜索
- 回车打开文件夹
- 点击条目右侧按钮编辑或删除

## 配置

- `traeFolderRemarks.displayPathStyle`
  - `relative`：列表中显示相对工作区路径
  - `absolute`：列表中显示绝对路径

示例配置见 [examples/settings.json](./examples/settings.json)

## 数据存储与隔离

备注数据存储在 VS Code/Trae 的 `workspaceState` 中：

- 同一工作区在窗口重载后自动还原
- 不同工作区之间天然隔离，不会串数据

## 安全说明

- Webview 使用严格 CSP（nonce）并对用户输入进行 HTML 转义，避免注入风险
- 不会采集或上报任何数据
